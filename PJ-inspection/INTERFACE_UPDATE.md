# 界面更新说明

## 更新内容

### 1. 双界面结构
程序现在包含两个独立的界面：

#### 🏠 工作台界面（默认显示）
- **图标**: HOME (房子图标)
- **位置**: 左侧导航栏第一个
- **内容**: 显示欢迎信息
  - 标题: "欢迎使用代码评审工具"
  - 提示: "请点击左侧'代码评审'图标开始使用"
- **状态**: 默认选中

#### 📄 代码评审界面
- **图标**: DOCUMENT (文档图标)
- **位置**: 左侧导航栏第二个（工作台下方）
- **内容**: 完整的代码评审功能
  - 文件树浏览
  - 文件内容预览（带行号）
  - 评审历史
  - 评审结果显示
  - 所有评审相关操作按钮

### 2. 代码文件预览增强
- ✅ 添加了行号显示功能
- ✅ 使用等宽字体（Consolas, Monaco, Courier New）
- ✅ 行号区域灰色背景便于区分
- ✅ 行号与代码内容自动同步滚动

## 文件修改清单

### main_window.py
1. **新增类**:
   - `LineNumberArea`: 行号显示区域
   - `CodeEditor`: 带行号的代码编辑器（继承QPlainTextEdit）

2. **UI初始化重构**:
   - `init_ui()`: 创建工作台界面（空白欢迎页）
   - `init_review_interface()`: 创建评审界面（原有功能）

3. **导航结构**:
   ```python
   # 工作台（默认）
   self.addSubInterface(self.workspace_widget, FluentIcon.HOME, '工作台')
   
   # 代码评审
   self.addSubInterface(self.review_widget, FluentIcon.DOCUMENT, '代码评审')
   ```

## 使用方式

1. **启动程序**: `python3 main.py`
2. **默认显示**: 工作台欢迎界面
3. **开始评审**: 点击左侧导航栏的"代码评审"图标
4. **返回工作台**: 点击左侧导航栏的"工作台"图标

## 技术实现

### 行号显示原理
1. **LineNumberArea**: 自定义Widget作为行号区域
2. **CodeEditor**: 
   - 继承`QPlainTextEdit`
   - 计算行号宽度（根据总行数动态调整）
   - 监听滚动和内容变化事件
   - 绘制行号（QPainter）
3. **同步机制**: 
   - `updateRequest`信号连接到行号更新
   - `blockCountChanged`信号调整宽度

### 界面分离优势
- ✅ 清晰的功能划分
