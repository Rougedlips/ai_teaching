# 代码评审工具

一个基于PyQt5的本地代码评审工具，支持调用大模型API对项目代码进行智能评审。

## 功能特性

- 📁 文件夹浏览和目录树展示
- 📄 文件内容预览
- 🤖 大模型API代码评审
- 📝 评审历史记录管理
- ⚙️ 自定义API配置和提示词

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```

## 使用说明

1. 点击"打开文件夹"选择要评审的项目目录
2. 在左侧目录树中浏览文件，点击文件可在中间区域预览内容
3. 点击右上方"开始评审"按钮，调用大模型进行代码评审
4. 评审结果会显示在右下方，并自动保存到历史记录
5. 点击底部历史记录可以查看以往的评审结果
6. 通过"设置"按钮配置API地址和自定义提示词

## 配置

首次使用需要在设置中配置：
- API地址：大模型API的接口地址
- API密钥：访问API所需的密钥
- 提示词：自定义代码评审的提示词模板
